# Jump Runner - Telegram Mini App

Jump Runner - это мобильная игра для Telegram, сочетающая в себе механики Dino Chrome и Flappy Bird. В этой игре персонаж должен прыгать, чтобы избегать препятствий, которые генерируются процедурно.

## Особенности игры

- **Процедурная генерация**: Препятствия, их типы и расположение генерируются случайным образом.
- **Адаптивный дизайн**: Игра оптимизирована для мобильных устройств и экранов смартфонов.
- **Прогрессивная сложность**: Скорость игры и частота появления препятствий постепенно увеличиваются.
- **Простое управление**: Прыжок осуществляется одним касанием экрана или кликом мыши.
- **Интеграция с Telegram**: Приложение готово для интеграции с Telegram WebApp API.

## Как играть

1. Нажмите кнопку "Начать игру" для старта.
2. Коснитесь экрана или нажмите на него, чтобы совершить прыжок.
3. Избегайте препятствий (как наземных, так и воздушных).
4. Набирайте очки, которые увеличиваются со временем.
5. Игра заканчивается при столкновении с препятствием.

## Запуск проекта

Для локального тестирования:

1. Откройте файл `index.html` в современном веб-браузере.
2. Для тестирования на мобильных устройствах используйте инструменты разработчика в браузере для переключения в мобильный режим.

Для интеграции с Telegram:

1. Разместите файлы проекта на веб-сервере с HTTPS.
2. Настройте Telegram Bot для использования игры как WebApp.
3. Укажите URL вашего приложения в настройках бота.

## Технические детали

- **Языки**: HTML5, CSS3, JavaScript (ES6+)
- **Совместимость**: Современные браузеры с поддержкой ES6
- **Разрешение**: Оптимизировано для мобильных устройств, но работает и на десктопе
- **Управление**: Сенсорный экран, мышь, клавиатура (пробел/стрелка вверх)
